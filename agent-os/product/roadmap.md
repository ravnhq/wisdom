# Product Roadmap

## Phase 1: MVP

- Build a cross-browser extension for Chrome, Chromium-based browsers, and Firefox using WXT.
- Replace the new tab page with a focused dashboard that shows a nature wallpaper, current time, saved user name, inspirational phrase, phrase explanation icon, and focus mode button.
- Ask for and store the user's name so the new tab page can feel personal.
- Add focus mode so users can intentionally enter a work-focused state.
- Support a configurable blocked-sites list that applies while focus mode is active.
- Ship with sensible default blocked sites for common distractions, with the exact default domains to be confirmed.
- Redirect blocked sites to a calm blocked-site page that shows an inspirational phrase instead of the distracting content.
- Load phrases, explanations, and related metadata from bundled JSON.
- Add quote controls for moving to the next quote and choosing whether a quote should appear again in the future.
- Include nature-style wallpapers, either bundled from provided assets or sourced from free-to-use images with proper licensing.
- Provide a polished, accessible UI with strong frontend practices.
